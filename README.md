# 麻雀の切り方定石トレーニング

麻雀で「何を切るか」を考える定石問題を出題する、Next.js + TypeScript の学習用ミニアプリです。

## 主な機能
- 局面・巡目・ドラ情報を確認しながらの切り牌問題
- 「答えを見る」で推奨打牌と解説を表示
- 前後の問題に移動して反復練習

## セットアップ
```bash
npm install
```

## 開発サーバー起動
```bash
npm run dev
```

## 本番ビルド
```bash
npm run build
```
